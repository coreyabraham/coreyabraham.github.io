Project Details

Game Engine: Unity

Platform: PC // Windows 7+ x64

Role(s): Systems and UI Programmer

Development Date: September the 8th, 2023 - December the 21st, 2023

Brief

Lights Out is a 3D top-down perspective Horror / Spooky Puzzle Game that was developed as part of the A.I.E Major Production Project for 1st Year Students!

It adapts the common fear of the dark during childhood and centers it's game ideas, art-style and game design to fit that mold.

This project had a time frame of around 5 weeks of pre-production and 6 weeks of production, creating a small game within that time-frame and making sure it properly functions for release.
